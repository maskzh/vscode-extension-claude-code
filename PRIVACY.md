# Privacy & Data Handling

## 密钥存储
- 在配置面板输入的 API Key 会保存在 VS Code 的 Secret Storage 中，不会写入明文文件或同步到 Git。
- 环境变量与命令配置存放在 VS Code 用户/工作区设置中，如包含敏感值可随时手动清空。

## 数据流
- 扩展仅在本机创建终端并执行命令（默认 `claude` 或自定义命令），不在扩展内部直接向网络发送请求。
- 与模型的网络通信由终端中运行的命令负责；请确保使用可信命令与安全网络环境。

## 遥测与日志
- 扩展未集成任何遥测、统计或远程日志上报。
- VS Code 自身可能记录常规扩展激活日志，但不包含您的 API Key 或对话内容。

## 清除与撤销
- 通过命令面板 `Claude Code Terminal: Configure Claude Code Terminals` 选择模型并留空提交，可清除对应的 API Key。
- 也可在 VS Code `设置 > 应用程序 > 安全性 > 秘密存储`（视版本路径不同）手动删除相关条目。

## 权限最小化建议
- 仅为需要的模型配置密钥，未配置的入口不会显示。
- 使用工作区范围设置时，确认工作区不被不可信人员共享。
